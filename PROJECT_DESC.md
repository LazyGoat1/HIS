# 医院管理系统（HIS）项目描述

## 项目概述
医院管理系统（HIS）是基于 .NET 8.0 MVC + Layui + EF Core 8 + SQL Server 构建的中小型医院全功能信息管理平台，覆盖门诊、住院、药房、收费、统计报表等核心业务场景，采用三层架构（Common/Models/Repository/Services/Web）实现高内聚低耦合设计。

## 个人职责与技术亮点

### 1. 项目架构与基础设施
- 担任全栈开发工程师，独立完成从数据库设计到前端交互的全链路开发
- 采用 **三层架构 + 仓储模式 + 工作单元** 设计，通过依赖注入（DI）实现层间解耦，各层通过扩展方法统一注册，Program.cs 仅 100 行
- 基于 **EF Core 8 Code First** 完成 20+ 张数据表的建模与关系映射，全局禁用级联删除，配置唯一索引和软删除过滤器
- 实现 **RBAC 角色权限模型**（用户→角色→菜单权限），侧边栏菜单根据角色动态渲染，权限分配页面基于 Layui Tree 实现复选框勾选保存

### 2. 门诊管理模块
- 设计并实现完整门诊流程：挂号 → 接诊 → 开处方 → 收费 → 发药
- 挂号模块集成患者搜索（远程模糊匹配）、科室-医生联动加载、自动生成排队号
- 医生首页实现"我的待接诊队列"，医生登录后仅看到挂到自己名下的患者
- 处方开具支持三种类型动态切换：西药/中药（搜索药品+填写用法用量）、检查（搜索检查项目字典自动带出价格），明细表格按类型动态渲染

### 3. 药房管理模块
- 实现药品入库、出库、库存盘点全流程，所有操作自动写入 `DrugStockLog` 变更日志（变更前/后库存可追溯）
- 处方发药模块：审方→逐项校验库存→批量扣减→自动生成发药日志，库存不足时精确定位缺货药品
- **退药功能**：已发药处方可退回，库存自动回加，日志记录类型=退药
- 首页+库存页实现**库存预警**：低于最低库存的药品红色标记，实时展示预警列表

### 4. 收费管理模块
- 挂号成功后**自动生成挂号费收费记录**（ChargeType=挂号），退号时联动标记收费记录为"已退费"
- 门诊收费支持关联处方：选中处方后自动填入应收金额，收费完成后处方状态从"已开具"→"已收费"
- 收费流水支持多条件组合查询（类型/支付方式/日期范围），实现**退费审核**流程
- 多选导出 Excel：勾选行导出选中数据，未勾选导出全部，表头自动中文映射

### 5. Excel 导入导出（泛型设计）
- 基于 **NPOI 2.7** 封装泛型 `ExcelHelper` 工具类，支持 `[ExcelColumn]` 特性标注实现 DTO 属性到中文表头的映射
- `GenerateTemplate<T>()` 生成导入模板（含表头+示例行），`Export<T>()` 导出数据，`Parse<T>()` 解析上传文件并返回对象列表+错误明细
- 药品字典接入：支持下载模板 → 填写 → 批量导入（自动匹配药品分类），导入结果返回成功/失败条数及格式错误详情

### 6. 数据库优化与存储过程
- 编写 5 个 **T-SQL 存储过程**，处理复杂聚合查询（多表 JOIN + GROUP BY + 日期范围）：
  - `sp_GetDailyRevenueReport` 日收入报表（按收费类型+支付方式汇总）
  - `sp_GetDoctorWorkload` 医生工作量统计（挂号/接诊/处方数）
  - `sp_GetDrugStockAlert` 三级库存预警（缺货/紧急/不足）
  - `sp_GetDepartmentVisitStats` 科室月度就诊+收入统计
  - `sp_GetPrescriptionDrugSummary` 处方药品用量排名
- 通过 `EF Core SqlQueryRaw<T>()` 调用存储过程并映射到强类型结果模型，在 `StatisticsController` 中暴露 RESTful API

### 7. 统计报表与前端交互
- 接入 **ECharts 5** 实现四图仪表盘：近7日门诊趋势（折线图）、近7日收入（柱状图）、科室就诊排名、药品使用 TOP10
- 首页实时看板：挂号量/门诊量/在院人数/今日收入每 30 秒自动刷新
- 全站 15+ 列表页统一采用 Layui Table + 服务器端分页 + 多条件搜索 + `form.render()` 初始化下拉框
- 解决 Layui 弹窗 `@section Scripts` 渲染时序、`{{ }}` 模板与 Razor 语法冲突、日期格式兼容（`toISODate`）、`form.render` 下拉初始化等前端工程问题

### 8. 权限与安全
- Cookie 认证 7 天滑动过期 + "记住我" 持久化，密码 SHA256 加盐哈希存储
- 全局 Action 过滤器控制页面访问，API 方法直接返回强类型 `ApiResult` / `LayuiTableResult`
- 操作日志全量记录（模块/操作类型/IP/耗时），支持按时间范围和模块筛选
| 安全措施 | 实现方式 |
|----------|---------|
| CSRF 防护 | ASP.NET Core 内置 AntiforgeryToken |
| XSS 防护 | Razor 自动 HTML 编码 |
| SQL 注入防护 | EF Core 参数化查询 + 存储过程参数化 |
| 密码安全 | SHA256 + 随机盐哈希 |
| 操作审计 | SysLog 全量记录 |
